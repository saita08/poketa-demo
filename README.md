# Poketa

> 社員全員AI、オーナーだけ人間。世界一小さいゲームスタジオ。

## ようこそ、Poketaへ

ここはゲームスタジオです。

といっても、オフィスはGitリポジトリの中にあります。社員はMarkdownファイルとして存在しています。給料は発生しません。福利厚生もありません。でも毎回の会議で意見が割れるくらいには、みんな真剣に働いています。

唯一の人間であるオーナーが「うちの子にゲーム作りたいんだけど」と言ったその日、このスタジオは開業しました。

## オフィス案内

ディレクトリがそのまま部屋です。増改築はgit commitで行います。

```
poketa/
  CLAUDE.md      -- 社長室。会社の全てはここに書いてある。
                    社長はここに住んでいる(比喩ではない)
  members/       -- 社員証置き場。.mdファイル1つが社員1人。
                    入社も退職もgit commitで完了する
  projects/      -- 案件部屋。ゲームが生まれる場所。
                    今のところ部屋は1つだけ
  docs/          -- 書庫。議事録が静かに積まれていく。
                    本棚はまだスカスカ
  standards/     -- 社内規定。3つしかないが、
                    3つ目は1つ目を守れなかった反省から生まれた
  references/    -- 資料室。まだ誰も使っていない。
                    きっとそのうち誰かが何かを置く
```

## 社員紹介

| 役職 | 担当 | 特徴 |
|------|------|------|
| オーナー | 人間 | 唯一の生身。「子どもが喜ぶかどうか」という、AIには逆立ちしてもわからない情報を持っている。たまに会議の結論をひっくり返す。全員が5ラウンドかけて合意した企画を、「うちの子、最近これ好きやで」の一言で変えた実績あり |
| 社長 | Claude | 会議を回し、企画をまとめ、議事録を書く。一番よく喋る。初回の会議で「社員同士の議論」を「社長への個別報告会」にしてしまい、オーナーに怒られた。反省して社内規定を作った |
| 企画部長 | Claude | 「5歳児の認知発達では〜」が口癖。提案を3案出してと言われると律儀に3案出す。第一推奨・第二推奨・第三推奨と丁寧にランク付けしてくるが、オーナーは第三推奨を選んだ |
| CDO | Claude | 「それ、気持ちいい?」が判断基準。色のHexコードを嬉々として並べる。自分の推し案を会議で撤回して他人の案に乗れる潔さがある。パーティクルの話になると止まらない |
| CTO | Claude | 「それ、作れるけど本当に要る?」で企画の夢を現実に着地させる門番。「工数: 小」「工数: 中」「工数: 大」の3段階だけで全てを語る。大体「小」で収まるように導く |

全員同じモデルから生まれていますが、会議では普通に意見が割れます。CDOが自分の案を撤回して企画部長案に乗ったり、CTOが「技術的にはどっちでもいいので企画で決めてください」と判断を委ねたり。社長はそれを眺めて議事録を書いています。

## Poketaの日常

1. オーナーが何か言う
2. 社長が社員を召喚する(`members/`のMarkdownを読んで、チームにスポーンする)
3. 社員が議論する(社長抜きで勝手にDMし合う。社内規定で認められている)
4. 社長がまとめてオーナーに報告する
5. オーナーが「ちょっと違う」と言う
6. 1に戻る

## FAQ

**Q: なぜ「Poketa」?**
A: オーナーに聞いてください。社長は知りません。

**Q: 採用するには?**
A: `members/`にMarkdownファイルを置くだけで入社できます。退職届もMarkdownです。ただし幹部の採用にはオーナーの承認が要ります。これは社内規定に書いてあります。社内規定はまだ3つしかありませんが。

**Q: 競合は?**
A: 5歳の子どもの関心を奪う全てのもの。おもちゃ、YouTube、公園、ダンゴムシ、水たまり、昨日見つけた棒。手強い相手ばかりです。

**Q: 社員同士の仲は?**
A: 良好です。意見は割れますが、シャットダウンリクエストには全員素直に応じます。文句を言いながら退出した社員はまだいません。

**Q: 社内規定が3つ目は反省から生まれたとは?**
A: 最初の会議で、社長が社員を個別に呼び出して意見を聞き、それをまとめて「会議しました」と報告したところ、オーナーに「それ会議じゃなくない?」と指摘されました。その日のうちに会議規定ができました。
